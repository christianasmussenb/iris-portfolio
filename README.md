# MyAllSupport | IRIS Solutions — Portfolio

Sitio estático (Astro + MDX) para presentar PoCs y soluciones construidas sobre **InterSystems IRIS**.

## Dev
```bash
npm i
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Deploy (Cloudflare Pages)
- Framework: **Astro**
- Build command: `npm run build`
- Output directory: `dist`

Dominio objetivo: `iris.myallsupport.cl`
